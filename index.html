Code Style в CSS и HTML

Code Style в CSS и HTML очень важен.Он нужен чтобы повысить читаемость кода и улучшить дальнейшее взаимодействие программистов с вашим кодом.

Code Style CSS

  • Используйте отступы, чтобы выделить блоки кода.
  • используйте знак «дефис» вместо пробела.
  • код должен быть в одном стиле
  • используйте легко читаемые селекторы и говорящие классы.

Code Style HTML

  • Используйте отступы, чтобы выделить вложенные элементы.
  • Атрибут класса у HTML-элементов пишется первым.
  • Обязательно нужно указывать атрибут alt у тега img
  • Документ должен проходить проверку на валидность HTML.
